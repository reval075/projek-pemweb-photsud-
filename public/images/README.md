# Memoforia image assets

Replace SVG placeholders with real photos (same filenames, `.jpg` or `.webp` preferred).

| Folder    | Purpose                          |
|-----------|----------------------------------|
| `hero/`   | Hero background / main visual    |
| `events/` | Philosophy, services cards       |
| `strips/` | Floating photostrip decorations  |
| `gallery/`| Gallery fallbacks                |

Update paths in `resources/js/constants/homeAssets.js` when switching extensions.
